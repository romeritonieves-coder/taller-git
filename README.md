# EduCampus LMS

EduCampus LMS es una plataforma educativa para la gestión de cursos, usuarios, evaluaciones y reportes académicos.

## Objetivo

Centralizar los procesos académicos de una institución educativa mediante una solución web modular.

## Público Objetivo

- Instituciones educativas (colegios, universidades, centros de formación)
- Docentes que imparten cursos en línea
- Estudiantes que requieren una plataforma de aprendizaje virtual
- Administradores académicos que gestionan programas y evaluaciones

## Tecnologías Principales

- **Frontend**: React.js, TypeScript
- **Backend**: Node.js, Express
- **Base de datos**: PostgreSQL
- **Herramientas**: Git, GitHub, Docker
